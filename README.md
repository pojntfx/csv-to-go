## Usage

```bash
% csv-to-go-generator -help
Usage of csv-to-go-generator:
  -in string
        Path to the CSV input file (default "data.csv")
  -out string
        Path of the generated output file (default "csv.go")
  -package string
        Package name of the generated output file (default "csv")
```
